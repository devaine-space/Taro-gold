# Taro22

Стартовый черный лендинг для сайта таро.

Что уже есть:
- hero-блок с верхним орнаментом;
- секция с вопросами и карточками;
- блок "Обо мне";
- блок с шагами работы;
- отзывы и финальный CTA;
- кликабельные символы и кнопки для дальнейшего редактирования.

Файлы проекта:
- `index.html`
- `styles.css`
- `script.js`

Сейчас это каркас, который можно дальше наполнять фото, текстами, формой записи и анимацией.
